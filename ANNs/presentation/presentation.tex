  \item Aprendizaje no supervisado

\begin{frame}{Enfoques actuales}
  
  \begin{itemize}
  \item Actualmente las redes neurales son poco usadas en juegos
    comerciales.
  \item En las versiones digitales de juegos de mesa como
    \emph{Othello}, \emph{Ajedrez}, \emph{Backgammon}, etc. es mas
    común el uso de redes neurales, por ser juegos de estrategia.
  \end{itemize}

  Los juegos de mesa suelen ser juegos más lentos con menos cambios de
  ambiente y mucha estrategia, mientras que en los juegos modernos hay
  muchos otros factores con que lidear aparte de la inteligencia
  artificial.
\end{frame}

\begin{frame}{Enfoques actuales}
  A pesar del poco uso, las redes neurales actualmente estan siendo
  utilizadas en el proceso de toma de decisiones de los agentes.
  \begin{figure}[htp]
    \centering
    \includegraphics[scale=0.3,angle=270]{media/decision-tree.png}
  \end{figure}
  Los agentes procesan lo recibido por sus sensores, y a través de una
  red entrenada (usualmente de forma \emph{offline}) toman acciones a
  partir de su output.\\

  Esto supone ahorro de código y complejidad de maquinas de estados y
  agentes basados en reglas.
\end{frame}

\begin{frame}{Aprendizaje en vivo (\emph{Online Learning})}

  \emph{Online Learning:} Entrenar la red neural con los datos
  recibidos por el jugador, una vez que el juego ya ha sido totalmente
  desarrollado.

  \begin{itemize}
  \item \textbf{Ganancias de desarrollar ésta técnica}
    \begin{itemize}
    \item Adaptación dinámica al comportamiento de los jugadores.
    \item Adaptación dinámica al nivel de juego de los jugadores.
    \item Puede ser usado para contrarrestar los jugadores que
      obtienen puntos en el juego haciendo trampa.
    \end{itemize}
  \end{itemize}
\end{frame}

\begin{frame}{Aprendizaje en vivo (\emph{Online Learning})}
  \begin{itemize}
  \item \textbf{Obstáculos para desarrollarla}
    \begin{itemize}
    \item Resultados impredecibles.
    \item Introducción dinámica de bugs en el juego.
    \item El entrenamiento puede ser requerido muchas veces para poder
      adaptar la red neural y balancearla a los requerimientos del
      juego.
    \item El entrenamiento puede llegar a ser muy lento.
    \end{itemize}
  \end{itemize}
\end{frame}

\begin{frame}{Aproximaciones centradas en el jugador}
  \begin{itemize}
  \item Aproximaciones de aprendizaje sobre el jugador en si, para
    buscar mejorar la ``Experiencia de juego''.
  \item Se busca usar el poder que tienen las redes neurales para
    reconocer patrones y utilizarlos en las actuaciones de los
    jugadores.
  \item Se busca hacer que los juegos puedan tener un rango mas ancho
    de preferencias de los usuarios.
  \end{itemize}
\end{frame}

\begin{frame}{Aproximaciones centradas en el jugador}
  Algunas aproximaciones que se pueden tomar son las siguiente:

  \begin{itemize}
  \item Usar los tiempos de juego de un jugador, sus reacciones, sus
    decisiones tomadas, estilo de juego, precision de sus disparos,
    promedio de vida, etc. como parte de los inputs de una red neural
    que se entrena \emph{offline} para determinar ciertos modelos de
    jugadores, y en base a esto ajustar el entorno de juego.

  \item Aprovechar la capacidad de las redes neurales para realizar
    ``Algorítmos de Clustering'' y agrupar a diferentes tipos de
    jugadores usando datos de su participacion dentro del juego,
    ajustandolos a ciertos perfiles de usuarios pre-establecidos.
  \end{itemize}
\end{frame}

\begin{frame}{Animación inteligente de personajes}
  Actualmente las animaciones que pretenden simular a los humanos
  se limitan a ciertas expresiones faciales o movimientos corporales
  previamente grabadas usando actores humanos pero ésto acarrea
  grandes costos humanos y poca precisión debido a la tecnología
  actual disponible para la captura de movimiento.
\newline
\newline
  El uso de redes neurales permite simular ciertos movimientos a
  partir de algunas capturas básicas.
\end{frame}
\begin{frame}{Animación inteligente de personajes}
  Ventajas del uso de esta técnica:
  \begin{itemize}
    \item Ahorro de tiempo de trabajo para los animadores.
    \item Mayor versatilidad en las animaciones.
    \item Permite la creación de animaciones en tiempo real, lo
      cual agrega realismo al juego.
  \end{itemize}
\end{frame}

\begin{frame}{Predicciones}
  A partir de una red neural debidamente entrenada podemos simular
  la reacción de un cierto ambiente ante unos estímulos y así posiblemente
  predecir la acción resultante.
  \newline
  \newline
  Tienen varias aplicaciones en otras áreas:
  \begin{itemize}
    \item Finanzas.
    \item Predicción del clima.
    \item Comsumo de energía eléctrica.
    \item Forecast de ventas.
    \end {itemize}
\end{frame}
\begin{frame}{Predicciones}
  En el área de juegos no son comunes pero pueden ser utilizados
  para: 
  \begin{itemize}
    \item En juegos de estrategia al decidir las acciones de un oponente
      computarizado prediciendo la próxima acción del jugador.
    \item En juegos de primera persona los "bots" pueden actuar en
      función del movimiento del jugador para contrariarlo.
  \end{itemize}
  La predicción ofrece la ventaja de que permite tener una respuesta
  no-determinística a una situación y por lo tanto permite la implementación
  de oponentes computarizados más interesantes y variados.
\end{frame}

\begin{frame}{Estudios a nivel de inteligencia humana}
  Consisten en simular mediante el uso de inteligencia artificial
  las acciones y decisiones que podrían ser tomadas por un humano.
  \newline
  \newline
  En los juegos se pueden usar las redes neurales para crear un jugador 
  virtual que permita el entrenamiento off-line de las demás redes del 
  sistema y que se pueda adaptar a cualquier juego donde sea configurado 
  lo cual permitiría que dicho entrenamiento pueda formar parte del
  perfil de un jugador para una plataforma de juego.
\end{frame}

\begin{frame}{Evolucionando redes neurales}
  El algorítmo más común para construir la topología de una red neural
  es \emph{backpropagation}, sin embargo existen otras aproximaciones
  que se pueden tomar para resolver este problema:

  \begin{center}
    \textbf{Algorítmos Evolutivos}
  \end{center}
\end{frame}

\begin{frame}{Evolucionando redes neurales}
  La idea principal es evolucionar los pesos de los inputs en las
  neuronas, utilizando como fitness una función sobre la salida de la
  red neural, buscando optimizar el resultado de la salida.
\end{frame}

\begin{frame}{Caso de estudio}
  \begin{center}
    \textbf{Ms. Pacman}
  \end{center}

  \begin{figure}[htp]
    \centering
    \includegraphics[scale=0.4]{media/mspacman.png}
  \end{figure}

  \textbf{Enfoque:} Optimizar el controlador de toma de decisiones del
  agente \emph{Pacman}.
\end{frame}

\begin{frame}{Caso de estudio}
  \textbf{¿Por qué Pacman?}

  \begin{itemize}
  \item \textbf{Problema:} Un agente \emph{Pacman} puede recibir un
    input muy grande y variado, según el estado del mundo.

  \item \textbf{Posible solución:} Se busca realizar una relajación
    del problema para ser atacado de forma más tratable pero con
    eficiencia.

  \item La red neural intentará reducir la complejidad de tomar
    decisiones con árboles de decisión o reglas pre-fabricadas.
  \end{itemize}
\end{frame}

\begin{frame}{Caso de estudio}
  El algorítmo evolutivo que entrena a la red neural buscará
  \textbf{optimizar} el score que el agente \emph{Pacman} puede
  lograr, y para esto usará:

  \begin{itemize}
  \item \textbf{Fitness:} el score que logra obtener el agente al
    final de un juego o el score average al final de una secuencia de
    juegos, utilizando la red neural como controlador de decisiones.

  \item \textbf{Individuos:} Secuencia de pesos que tienen las
    neuronas de la red neural.
  \end{itemize}
\end{frame}
